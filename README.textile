h1. Vim Todo

vim-todo provides a simple todo list similar to Textmate's todo.bundle.

See the "Vim doc":http://github.com/svenfuchs/vim-tree/blob/master/doc/vim-todo.txt for details.

h2. Installation

Requires "vim-layout":http://github.com/svenfuchs/vim-layout and assumes a "pathogen":http://github.com/tpope/vim-pathogen style setup where both vim-layout and vim-todo are subdirectories of your runtimepath.
